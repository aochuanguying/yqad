## 需求
### 需求：脚本执行功能
系统必须允许用户在 AutoJS 配置页面手动执行任意脚本，并提供执行状态反馈。

#### 场景：用户点击执行按钮
- **当** 用户点击某个脚本旁边的"立即执行"按钮
- **那么** 系统调用 AutoJS API 的 `/api/execute` 接口执行该脚本

#### 场景：脚本执行成功
- **当** AutoJS API 返回执行成功响应
- **那么** 显示成功提示并将按钮恢复为可点击状态

#### 场景：脚本执行失败
- **当** AutoJS API 返回错误响应或请求超时
- **那么** 显示错误提示并将按钮恢复为可点击状态

#### 场景：执行中防止重复点击
- **当** 脚本正在执行中
- **那么** 禁用执行按钮并显示加载动画
