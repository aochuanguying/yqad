## MODIFIED Requirements

### 需求: 系统必须自动选择帖子并发布AI生成的评论
系统必须从帖子列表中智能选择适合评论的帖子，生成评论内容并发布。评论发布时必须传递帖子的 contentType 以满足真实 API 的参数要求。

#### 场景: 选择评论目标帖子
- **当** 自动评论任务触发
- **那么** 系统必须从最新帖子中选择尚未评论过的帖子作为评论目标

#### 场景: 生成并发布评论
- **当** 确定评论目标帖子后
- **那么** 系统必须调用AI内容生成模块生成评论，然后调用发布评论API发布

#### 场景: 发布评论时传递帖子类型
- **当** 调用 publishComment 发布评论
- **那么** 系统必须将帖子的 contentType（INFORMATION/DYNAMIC/ARTICLE/NOTES）作为参数传递给 API 客户端

#### 场景: 每日评论数量可配置
- **当** 执行每日评论任务
- **那么** 系统必须根据配置的每日评论数量上限（默认3条）执行评论
