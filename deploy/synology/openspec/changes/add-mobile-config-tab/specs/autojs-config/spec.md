## 修改需求

### 需求：AutoJS 配置页面位置调整

AutoJS 配置页面必须从独立的 `/autojs-config.html` 移动到"手机配置"页签下的子功能。

#### 场景：访问手机配置页面的 AutoJS 配置
- **当** 用户在手机配置页面点击"AutoJS 配置"子功能
- **那么** 显示与原 `autojs-config.html` 相同的功能界面

#### 场景：保留原有功能
- **当** AutoJS 配置页面被集成到手机配置页签下
- **那么** 所有原有功能（启用状态、服务器地址、发帖脚本、脚本列表管理、测试连接）保持不变

## 移除需求

### 需求：独立 AutoJS 配置页面访问

**Reason**: AutoJS 配置已整合到"手机配置"页签下，不再需要独立页面访问
**Migration**: 通过首页"手机配置"页签访问 AutoJS 配置功能
