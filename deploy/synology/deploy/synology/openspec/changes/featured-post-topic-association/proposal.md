## 为什么

当前精华帖发布功能在生成高质量内容后，发布时没有自动关联相关话题，导致帖子曝光度不足。虽然系统已有热门话题 API 和 AI 语义匹配能力，但未集成到发帖流程中。需要在精华帖发布时自动匹配并关联话题，提升帖子可见度和互动率。

## 变更内容

- **新增**：在精华帖发布流程中集成话题自动关联功能
- **新增**：基于帖子标题和内容进行 AI 语义话题匹配
- **修改**：`AutoPostService` 的 `postWithTopic` 方法，在发帖成功后关联匹配的话题
- **修改**：`RealApiClient` 增加话题关联 API 调用方法

## 功能 (Capabilities)

### 新增功能
- `topic-association`: 精华帖发布时自动关联话题功能，包括获取热门话题、AI 语义匹配、调用关联 API

### 修改功能
- `auto-post`: 发帖服务需要集成话题关联逻辑，在发帖成功后自动调用话题关联

## 影响

- **代码影响**：
  - `src/services/auto-post.ts`：发帖成功后增加话题关联调用
  - `src/api/real-client.ts`：新增话题关联 API 方法
  - `src/services/topic-matcher.ts`：已有话题匹配逻辑，需确保返回格式符合关联需求
- **API 依赖**：需要调用社区话题关联接口（需确认具体 API 端点）
- **发帖流程**：精华帖发布流程增加话题关联步骤（失败不影响发帖成功）
