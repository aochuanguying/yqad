## 修改需求

### 需求: 系统必须自动生成并发布优质帖子
系统必须定时基于社区热点话题生成高质量帖子并发布到社区。发帖主题配置中，素材附件字段必须通过文件选择器（而非手动路径输入）指定，配置值存储的是相对于素材库根目录的文件路径。

#### 场景: 基于热点生成帖子
- **当** 自动发帖任务触发
- **那么** 系统必须从内容分析结果中选取热点话题，调用AI生成包含标题和正文的帖子

#### 场景: 发布帖子
- **当** 帖子内容生成完毕
- **那么** 系统必须调用发帖API将帖子发布到社区

#### 场景: 每日发帖数量可配置
- **当** 执行每日发帖任务
- **那么** 系统必须根据配置的每日发帖数量上限（默认1篇）执行发帖

#### 场景: 发帖主题配置使用文件选择器指定素材
- **当** 用户在 Web 管理界面配置发帖主题并需要附加素材
- **那么** 素材字段必须通过文件选择器从素材库中选取，配置中存储相对路径（如 `images/banner.jpg`），禁止存储绝对路径或手动输入任意字符串

#### 场景: 发帖时读取素材文件
- **当** 发帖任务执行且主题配置中包含素材相对路径
- **那么** 系统必须将相对路径与素材库根目录（`materials.basePath`）拼接得到完整路径后读取文件
