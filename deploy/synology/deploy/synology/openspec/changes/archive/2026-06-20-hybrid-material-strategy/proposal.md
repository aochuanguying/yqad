# 自由发帖模式素材混合策略

## 为什么

当前自由发帖功能存在以下问题：

1. **素材来源单一**：自由发帖只使用互联网参考素材，本地素材库中的高质量素材未被充分利用
2. **素材利用率低**：本地拍摄的优质素材（如旅行照片、实车图片）长期闲置，造成资源浪费
3. **缺乏智能匹配**：无法根据互联网参考的标题智能匹配相关的本地素材
4. **素材优先级不明确**：没有建立本地素材与网络素材的优先级策略
5. **素材质量参差不齐**：缺乏素材质量评估机制，难以优先使用高质量素材
6. **素材使用不可追溯**：没有记录素材的使用情况，无法分析素材利用效果

虽然已有基础的材料库管理和发帖功能，但缺乏素材混合使用机制，导致本地素材利用率低。需要在合规性检查（第一步优化）和主题多样化（第二步优化）的基础上，进一步增强素材混合策略，提升素材利用率和发帖质量。

## 变更内容

- **新增**：本地素材优先级配置（可配置本地优先/网络优先/混合模式）
- **新增**：基于互联网参考标题的本地素材智能匹配
- **新增**：素材混合策略（本地素材 + 网络素材组合使用）
- **新增**：素材质量评估和选择（优先使用高质量素材）
- **新增**：素材使用记录追溯（记录来源、使用情况、使用次数）
- **修改**：`auto-post.ts` 的素材选择逻辑，集成混合策略
- **修改**：`content-generator.ts` 的素材引用逻辑，支持混合素材使用
- **修改**：`topics-service.ts` 的素材管理，增加质量评估和追溯

## 功能 (Capabilities)

### 新增功能
- `material-priority-config`: 本地素材优先级配置（本地优先/网络优先/混合模式）
- `smart-material-matching`: 基于互联网参考标题的本地素材智能匹配
- `hybrid-material-strategy`: 素材混合策略（本地素材 + 网络素材组合使用）
- `material-quality-scoring`: 素材质量评估和选择
- `material-usage-tracking`: 素材使用记录追溯

### 修改功能
- `auto-post`: 发帖服务需要集成素材混合策略到素材选择流程
- `content-generator`: 内容生成器需要支持混合素材的引用和使用
- `topics-service`: 素材服务需要支持质量评估和使用追溯

## 技术实现要点

### 1. 本地素材优先级配置
- 定义三种优先级模式：
  - **本地优先**：优先使用本地素材，不足时补充网络素材
  - **网络优先**：优先使用网络参考素材，不足时补充本地素材
  - **混合模式**：按配置比例混合使用本地和网络素材
- 支持动态配置和热更新

### 2. 智能素材匹配
- 基于互联网参考标题提取关键词
- 使用关键词匹配本地素材的标签、描述、元数据
- 支��语义相似度匹配（可选 AI 增强）
- 返回匹配度评分，优先选择高匹配度素材

### 3. 素材混合策略
- 定义混合比例配置（如本地 70% + 网络 30%）
- 支持按发帖类型调整混合比例
- 确保素材之间的协调性和一致性

### 4. 素材质量评估
- 定义质量评分维度：
  - **清晰度**：分辨率、噪点、模糊度
  - **构图**：主体突出、画面平衡
  - **光线**：曝光适当、对比度
  - **相关性**：与主题的相关程度
  - **新鲜度**：素材使用次数（越少越新）
- 综合评分，优先使用高质量素材

### 5. 素材使用追溯
- 记录每次素材使用的详细信息：
  - 使用时间
  - 关联的发帖 ID
  - 素材来源（本地/网络）
  - 素材质量评分
  - 匹配度评分
- 支持查询和分析素材使用历史

## 验收标准

1. **优先级配置**：
   - 支持三种优先级模式切换
   - 配置生效正确
   - 发帖素材选择符合优先级策略

2. **智能匹配**：
   - 基于标题的匹配准确率 > 70%
   - 匹配过程在 2 秒内完成
   - 匹配结果有明确的评分

3. **素材混合**：
   - 混合比例符合配置
   - 本地素材使用率提升 50% 以上
   - 素材组合协调一致

4. **质量评估**：
   - 质量评分维度完整
   - 优先使用高质量素材
   - 质量评分与人工评估一致性 > 80%

5. **使用追溯**：
   - 使用记录完整准确
   - 支持查询和统计
   - 数据持久化正常

## 风险和依赖

- **依赖**：建立在第一步合规性检查和第二步主题多样化的基础上
- **风险**：智能匹配可能不准确，影响素材选择效果
- **缓解**：提供手动调整和反馈机制，持续优化匹配算法
