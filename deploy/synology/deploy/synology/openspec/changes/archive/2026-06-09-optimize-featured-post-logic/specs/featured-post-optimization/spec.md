## 新增需求

### 需求：系统必须优化精华帖标题生成策略
系统必须在生成精华候选帖时应用标题优化策略，确保标题符合官方精华帖攻略要求。

#### 场景：标题字数控制
- **当** 系统生成精华候选帖标题时
- **那么** 标题字数必须控制在 10-20 字之间

#### 场景：标题包含热点关键词
- **当** 系统生成精华候选帖标题时
- **那么** 标题必须包含至少 1 个热点关键词（如车型、功能、场景相关词汇）

#### 场景：标题使用吸睛句式
- **当** 系统生成精华候选帖标题时
- **那么** 标题必须使用以下句式之一：提问式、数据式、痛点式

### 需求：系统必须优化精华帖内容结构
系统必须在生成精华候选帖正文时应用结构化约束，确保内容逻辑清晰、层次分明。

#### 场景：开头引入
- **当** 系统生成精华候选帖正文时
- **那么** 开头必须使用提问、数据或痛点引入（1-2 句）

#### 场景：分段逻辑
- **当** 系统生成精华候选帖正文时
- **那么** 正文必须包含至少 2 个小标题或编号列表

#### 场景：结尾互动引导
- **当** 系统生成精华候选帖正文时
- **那么** 结尾必须包含互动引导语（邀请评论、点赞、收藏）

### 需求：系统必须优化精华帖图片选择策略
系统必须在选择精华候选帖图片时应用多角度、高质量约束，确保图片符合精华评选标准。

#### 场景：图片数量目标
- **当** 系统为精华候选帖选择图片时
- **那么** 图片数量目标必须为 6-9 张（高于最低 4 张要求）

#### 场景：多角度覆盖
- **当** 系统为精华候选帖选择图片时
- **那么** 必须优先选择覆盖外观、内饰、细节、场景等多角度的图片组合

#### 场景：图片质量要求
- **当** 系统为精华候选帖选择图片时
- **那么** 必须优先选择描述中包含"高清"、"实拍"、"细节"等关键词的素材

### 需求：系统必须优化精华帖生成提示词
系统必须在 AI 生成提示词中融入精华帖创作攻略的完整要求，指导 LLM 产出符合精华标准的内容。

#### 场景：提示词包含标题要求
- **当** 系统调用 AI 生成精华候选帖时
- **那么** 提示词必须明确标题要求（10-20 字、热点词、吸睛句式）

- **当** 系统调用 AI 生成精华候选帖时
