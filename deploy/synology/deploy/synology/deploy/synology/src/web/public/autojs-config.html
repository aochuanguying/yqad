API Token 说明

AutoJS API 使用本服务生成的发帖 API Token进行认证。
请确保已在 🔑 API Token 页面 生成 Token。

相对于 AutoJS 脚本目录的文件名，默认：audi_post.js

📖 使用说明

  • 启用后，点击"立即发帖"按钮会调用 AutoJS API 执行脚本
  • 脚本会在手机上运行，自动完成发帖流程
  • 发帖成功后，脚本会回调本服务更新日志状态
  • 如果脚本执行异常未回调，30 分钟后日志会自动标记为失败
