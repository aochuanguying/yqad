## 🔍 问题诊断

### 为什么又回到 300011？

1. **第一次 300011**: 之前频繁测试触发风控
2. **第二次 -100**: Cookie 自然过期（6 小时后）
3. **当前 300011**: 使用新 Cookie 测试，但**风控限制仍未解除**

### 风控机制分析

小红书的风控系统：
- **触发条件**: 高频 API 调用、非正常请求模式
- **限制对象**: 账号（不是 Cookie）
- **持续时间**: 通常 24-48 小时，严重情况可能更长
- **表现**: 错误码 300011，即使使用新 Cookie 也无效
