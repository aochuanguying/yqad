## 核心发现

### ✅ 正确的技术实现

1. **搜索 API**
   - URL: `https://sou.api.autohome.com.cn/v1/search`
   - 无需 Cookie 即可调用
   - 返回 `type: "card"` 的帖子数据

2. **正文获取**
   - 使用 Playwright 打开帖子 URL
   - 选择器：`.fn-main .post` ✅ （经过验证）
   - 旧选择器 `.fn-container .fn-main .post` ❌ （不正确）

3. **技术栈**
   - 搜索 API：HTTP requests
   - 正文获取：Playwright + Chromium
   - 反检测：stealth.min.js

### ❌ 已删除的错误信息

1. **需要 Cookie 的误解**
   - 旧文档提到需要 F12 抓包获取 Cookie
   - 实际情况：搜索 API 无需 Cookie
   - 删除：`AUTOHOME_CONFIG_SETUP.md`

2. **旧的正文提取逻辑**
   - 旧逻辑：遍历整个页面文本，用关键词过滤
   - 新逻辑：直接使用 `.fn-main .post` 选择器
   - 代码已简化 ~70 行

3. **过时的 URL 格式**
   - 旧 URL：`club.autohome.com.cn/bbs/forum-{id}-1.html`（404）
   - 新 URL：`club.autohome.com.cn/bbs/thread-{threadid}-{page}-1.html`
