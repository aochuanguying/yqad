## 📋 一、功能概述

实现了通过 Playwright 访问小红书详情页并提取完整内容的功能，解决了之前只有搜索功能无法获取详情的痛点。

### 技术方案

**核心思路**: 搜索 + Playwright 详情页爬取

1. **搜索阶段**: 使用 xhshow 签名 + requests 获取笔记列表（包含 note_id 和 xsec_token）
2. **详情获取**: 使用 Playwright 无头浏览器访问详情页，通过 JavaScript 提取内容

import { getXiaohongshuNoteDetail } from './src/services/internet-reference-service';

### 3.3 方式三：直接调用搜索服务

```typescript
import { XiaohongshuSearch } from './src/services/internet-search/xiaohongshu-search';

async function test() {
  const xiaohongshu = new XiaohongshuSearch();
  
  // 1. 先搜索
  const searchResults = await xiaohongshu.search(['汽车评测'], 3);
  
  // 2. 获取第一条笔记的详情
  if (searchResults.length > 0) {
    const noteId = searchResults[0].url?.split('/').pop() || '';
    const detail = await xiaohongshu.getNoteDetail(noteId);
    
    console.log('详情:', detail.data);
  }
}
```

### 5.2 Cookie 处理

- 需要完整的 Cookie（包含 a1, web_session, id_token 等）
- Cookie 会从环境变量或配置文件中读取
- 支持通过 `XIAOHONGSHU_COOKIE` 环境变量覆盖

## ⚠️ 六、注意事项

### 6.1 Cookie 有效期

- Cookie 会过期（通常几天到几周）
- 过期后需要重新获取
- 建议实现自动检测机制

### 6.2 请求频率

- 避免高频请求（建议间隔 3-5 秒）
- 长时间高频使用可能导致 Cookie 失效
- 已实现随机休眠机制

### 6.3 性能考虑

- Playwright 启动需要时间（约 2-3 秒）
- 页面加载需要时间（约 3-5 秒）
- 总耗时约 5-10 秒/篇笔记

### 6.4 错误处理

常见错误及解决方案：

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 笔记暂时无法浏览 | Cookie 权限不足或笔记被删除 | 更换 Cookie 或笔记 ID |
| 无法从页面提取内容 | 页面结构变化 | 更新 JavaScript 选择器 |
| Playwright 错误 | 浏览器启动失败 | 检查依赖和权限 |
| 获取详情超时 | 网络问题或页面加载慢 | 增加超时时间（当前 60 秒） |
