**文件**:
- `src/services/post-retry-service.ts` - 新建重试服务
- `src/services/auto-post.ts` - 集成重试逻辑

**效果**:
- 自动重试失败的网络请求
- 指数退避策略（1 分钟、2 分钟、4 分钟...）
- 最大重试 3 次
- Redis 队列管理

**重试策略**:
```typescript
delay = baseDelay * 2^(retryCount-1)
// 第 1 次：1 分钟
// 第 2 次：2 分钟
// 第 3 次：4 分钟
```
