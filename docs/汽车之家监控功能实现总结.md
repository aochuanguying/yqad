# 汽车之家页面结构监控功能实现总结

## 📋 功能概述

**实现时间**: 2026-06-28  
**功能目标**: 当汽车之家页面结构变更导致选择器失效时，自动在配置页面显示警告提示

```sql
ALTER TABLE network_post_config 
ADD COLUMN autohome_selector_warning TEXT DEFAULT NULL

AFTER autohome_enabled;
```

### 4. Python 脚本

**检测逻辑** (`scripts/test_autohome.py`)：

```python
# 检测选择器失效
if not content_element:
    print(f"⚠️ 警告：选择器 '.fn-main .post' 未找到元素")
    
    # 发送警告到 API（异步，不阻塞）
    try:
        import urllib.request
        import json
        api_url = "http://localhost:8080/api/network-post-config/autohome-warning"
        warning_data = json.dumps({
            "warning": "页面结构可能已变更...",
            "timestamp": datetime.now().isoformat()
        }).encode('utf-8')
        
        req = urllib.request.Request(api_url, data=warning_data, method='POST')
        req.add_header('Content-Type', 'application/json')
        urllib.request.urlopen(req, timeout=5)  # 不等待响应
        print(f"✓ 警告已发送到配置页面")
    except Exception as api_error:
        print(f"⚠️ 发送警告到 API 失败：{api_error}")
    
    # 尝试 fallback 选择器
    fallback_selectors = [".post", ".thread-content", ...]
```

**特点**：
- ✅ 异步发送，不阻塞主流程
- ✅ 超时 5 秒，避免长时间等待
- ✅ 失败不影响正文获取
- ✅ 自动降级到 fallback 选择器

### 5. 前端页面

**警告横幅** (`src/web/public/index.html`)：

```html
<div id="autohome-warning-box" class="hidden p-4 bg-amber-50 border border-amber-200 rounded-md">
  <p class="text-sm text-amber-800">
    <strong>⚠️ 警告：</strong>
    <span id="autohome-warning-text"></span>
  </p>
  <p class="text-xs text-amber-700 mt-2">
    💡 汽车之家页面结构可能已变更，已自动尝试其他选择器。
  </p>
  <button onclick="clearAutohomeWarning()">✓ 我知道了</button>
</div>
```

**JavaScript 函数**：

```javascript
// 加载警告
async function loadAutohomeWarning() {
  const resp = await fetch('/api/network-post-config/autohome-warning');
  const data = await resp.json();
  
  if (data.hasWarning && data.warning) {
    document.getElementById('autohome-warning-text').textContent = data.warning;
    document.getElementById('autohome-warning-box').classList.remove('hidden');
  }
}

// 清除警告
async function clearAutohomeWarning() {
  const resp = await fetch('/api/network-post-config/autohome-warning', {
    method: 'POST',
    body: JSON.stringify({ clear: true })
  });
  
  const data = await resp.json();
  if (data.success) {
    document.getElementById('autohome-warning-box').classList.add('hidden');
  }
}
```

**页面加载**：
```javascript
// 在网络发帖配置页面加载时检查警告
setTimeout(() => loadAutohomeWarning(), 200);
```

2. 用户打开配置页面
