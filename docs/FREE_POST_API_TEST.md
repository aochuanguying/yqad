# 自由发帖 API 测试指南

本文档介绍如何直接调用 API 执行自由发帖测试。

## 测试文件说明

### 1. 单元测试文件
- **位置**: `src/__tests__/free-style-post-api.test.ts`
- **用途**: 使用 Jest 框架进行单元测试
- **测试场景**:
  - 自由发帖模式（互联网参考）
  - 精华帖模式的自由发帖
  - 互联网参考素材获取和 AI 改写
  - 完整发帖流程
  - 并发控制
  - 错误处理和降级
  - 发帖日志记录

### 2. 独立测试脚本
- **位置**: `scripts/test-free-post-api.ts`
- **用途**: 手动运行测试脚本，直接调用 API 发帖
- **特点**: 实时输出详细结果，适合调试和验证

## 测试参数说明

### performDailyPosts 参数

```typescript
async performDailyPosts(
  count: number,           // 发帖数量
  mode: PostingMode,       // 发帖模式：'normal' | 'featured'
  isManual: boolean        // 是否手动触发
): Promise<PostResult[]>
```

### PostingMode 模式

- **normal**: 普通帖（最少 100 字，图片数量不限）
- **featured**: 精华帖（最少 300 字，最少 3 张图片）

## 常见问题排查

### Q1: 测试失败，提示"互联网参考查询未返回结果"

**原因**: 互联网参考服务可能未正确初始化或网络问题

**解决方案**:
1. 检查网络连接
2. 确认小红书、微博等平台的 API 可访问
3. 检查 `internet-reference-service.ts` 的配置

### Q2: 测试失败，提示"Token 无效"

**原因**: API Token 配置问题

**解决方案**:
1. 检查配置文件 `config/local.yaml`
2. 确认 API Token 已正确配置
3. 使用 `GET /api/token/full` 获取完整 Token

### Q3: 发帖成功但图片数量为 0

**原因**: 图片上传失败或没有匹配到素材

**解决方案**:
1. 检查 CDN 配置
2. 确认本地素材库存在有效素材
3. 查看日志中的图片选择过程

### Q4: 并发测试失败

**原因**: 发帖任务正在执行中

**解决方案**:
1. 等待当前任务完成
2. 检查 `isPostTaskRunning` 标志位
3. 查看 `posts-routes.ts` 中的并发控制逻辑

## 监控和日志

### 查询发帖日志

```typescript
import { postLoggingService } from '../src/services/post-logging-service';

// 查询日志列表
const logs = await postLoggingService.query({
  page: 1,
  limit: 20,
  postType: 'free',  // 'free' | 'topic'
  triggerType: 'manual',  // 'manual' | 'auto'
});

// 查询单条日志详情
const log = await postLoggingService.getDetail(logId);

// 获取统计信息
const stats = await postLoggingService.getStats();

// 获取性能指标
const metrics = await postLoggingService.getPerformanceMetrics();
```

### 性能指标

监控以下关键指标：

1. **发帖成功率**: `success_count / total_count`
2. **平均发帖耗时**: `sum(duration) / count`
3. **合规检查通过率**: `passed_count / total_count`
4. **图片上传成功率**: `upload_success_count / total_uploads`
5. **降级发生率**: `downgrade_count / total_posts`

## 测试环境要求

### 必需组件

- ✅ Node.js (v18+)
- ✅ MySQL 数据库
- ✅ Redis 缓存
- ✅ ChromaDB 向量数据库
- ✅ 奥迪 API 访问权限

### 可选组件

- ⚠️ 互联网参考服务（小红书、微博等）
- ⚠️ CDN 图片上传服务
- ⚠️ AI 服务（OpenAI/DeepSeek/Ollama）

## 注意事项

1. **测试频率**: 避免频繁运行测试，以免触发平台频率限制
2. **测试数据**: 测试会产生真实的发帖记录，请注意清理
3. **Token 安全**: 不要将 Token 提交到版本控制系统
4. **并发控制**: 同时只能运行一个发帖任务
5. **日志清理**: 定期清理过期的发帖日志（建议保留 30 天）
