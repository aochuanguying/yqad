## 互联网参考服务详解

### 1. 服务职责

**文件位置**: [`src/services/internet-reference-service.ts`](file:///Users/mac/Documents/workspace/krio/yqad/src/services/internet-reference-service.ts)

互联网参考服务是自由发帖模式的核心，负责：

1. **多平台搜索**: 从小红书、微博、知乎、汽车之家等平台获取参考素材
2. **频率限制控制**: 每小时最多查询 N 次，避免触发平台反爬
3. **平台轮询策略**: 自动切换不同平台，分散请求压力
4. **去水印处理**: 去除下载图片的水印（预留接口）

### 2. 核心函数

#### 2.1 频率限制检查

```typescript
export async function canQuery(): Promise<boolean> {
  // 1. 检查服务是否启用
  const config = await getInternetReferenceStorage().getConfig();
  if (!config || !config.enabled) {
    return false;
  }

  // 2. 重置计数器（每小时）
  const now = Date.now();
  if (now - lastResetTime > 3600000) {
    queryCount = 0;
    lastResetTime = now;
  }

  // 3. 检查频率限制
  const rateLimit = config.rateLimitPerHour || 10;
  if (queryCount >= rateLimit) {
    logger.warn(`互联网参考查询频率超限：${queryCount}/${rateLimit} 次/小时`);
    return false;
  }

  return true;
}
```

**关键点**:
- 使用内存计数器 `queryCount` 跟踪查询次数
- 每小时自动重置计数器
- 从数据库读取配置，支持动态调整

#### 1.2 平台轮询策略

```typescript
async selectNextPlatform(): Promise<ISearchPlatform | null> {
  const platforms = await this.getAvailablePlatforms();
  
  if (platforms.length === 0) {
    logger.warn('没有可用的搜索平台');
    return null;
  }
  
  if (platforms.length === 1) {
    return platforms[0];
  }
  
  // 轮询策略：选择与上次不同的平台
  if (this.lastUsedPlatform) {
    const otherPlatforms = platforms.filter(
      p => p.getPlatformName() !== this.lastUsedPlatform
    );
    
    if (otherPlatforms.length > 0) {
      const sortedPlatforms = await this.sortByPriority(otherPlatforms);
      const selected = sortedPlatforms[0];
      
      this.lastUsedPlatform = selected.getPlatformName();
      return selected;
    }
  }
  
  // 默认返回优先级最高的平台
  const sortedPlatforms = await this.sortByPriority(platforms);
  const selected = sortedPlatforms[0];
  
  this.lastUsedPlatform = selected.getPlatformName();
  return selected;
}
```

**轮询逻辑**:
1. 获取所有可用平台（从数据库读取启用状态）
2. 如果只有一个平台，直接返回
3. 如果有多个平台，优先选择与上次不同的平台
4. 按优先级排序，选择优先级最高的平台
5. 记录上次使用的平台，用于下次轮询

#### 1.3 降级搜索

```typescript
async search(keywords: string[], maxResults: number = 5): Promise<SearchResult[]> {
  // 1. 选择平台
  const platform = await this.selectNextPlatform();
  
  if (!platform) {
    logger.warn('没有可用平台，返回空结果');
    return [];
  }
  
  try {
    // 2. 执行搜索
    const results = await platform.search(keywords, maxResults);
    return results;
    
  } catch (error) {
    logger.error(`平台 ${platform.getPlatformDisplayName()} 搜索失败`);
    
    // 3. 降级：尝试其他平台
    const fallbackResults = await this.searchWithFallback(
      keywords, 
      maxResults, 
      platform.getPlatformName()
    );
    
    return fallbackResults;
  }
}

private async searchWithFallback(
  keywords: string[], 
  maxResults: number, 
  excludePlatform: string
): Promise<SearchResult[]> {
  const platforms = await this.getAvailablePlatforms();
  const fallbackPlatforms = platforms.filter(
    p => p.getPlatformName() !== excludePlatform
  );
  
  // 4. 逐个尝试其他平台
  for (const platform of fallbackPlatforms) {
    try {
      const results = await platform.search(keywords, maxResults);
      if (results.length > 0) {
        return results;  // 有结果立即返回
      }
    } catch (error) {
      logger.warn(`平台 ${platform.getPlatformDisplayName()} 降级搜索失败`);
      // 继续尝试下一个平台
    }
  }
  
  logger.warn('所有平台降级搜索失败，返回空结果');
  return [];
}
```

**降级策略**:
1. 首选平台失败后，自动尝试其他平台
2. 按优先级顺序尝试
3. 一旦某个平台返回结果，立即返回
4. 所有平台都失败，返回空结果

#### 2.2 互联网参考模式

```typescript
private async tryInternetReferenceMode(
  mode?: PostingMode, 
  triggerType: 'auto' | 'manual' = 'auto'
): Promise<PostResult> {
  // 1. 读取配置
  const [postConfig, featuredConfig] = await Promise.all([
    getPostConfigStorage().getConfig(),
    getFeaturedPostingStorage().getConfig(),
  ]);
  const featuredEnabled = featuredConfig?.enabled ?? false;
  const minImages = featuredConfig?.minImages || 3;

  // 2. 检查频率限制
  if (!canQuery()) {
    logger.error('互联网参考查询频率超限，无法发帖');
    return { 
      success: false, 
      error: '互联网参考查询频率超限', 
      source: 'free' 
    };
  }

  // 3. 查询互联网参考素材
  const references = await search();
  
  // 4. 无结果降级处理
  if (!references || references.length === 0) {
    logger.warn('互联网参考查询未返回结果，使用 AI 直接生成内容');
    return await this.fallbackToAIGeneration(mode, featuredEnabled);
  }

  // 5. AI 改写生成
  const globalPrompt = loadGlobalPrompt() ?? undefined;
  const recentTopics = await this.getRecentTopics(postConfig?.avoidRepeatDays || 7);
  const topic = references[0].title || '奥迪用车分享';

  const generated = await this.generatePostWithMinChars(
    topic,
    recentTopics,
    undefined,
    {
      globalPrompt,
      referenceTexts: references,  // 传入参考素材
      mode: featuredEnabled ? 'featured' : 'normal',
    },
    featuredEnabled ? (featuredConfig?.minContentChars || 300) : 100,
    featuredEnabled ? (featuredConfig?.maxGenerateRetries || 0) : 0
  );

  // 6. 抄袭检测
  if (detectPlagiarism(generated.content, references)) {
    logger.error('生成内容与参考素材存在抄袭嫌疑，无法发帖');
    return { 
      success: false, 
      error: '生成内容存在抄袭嫌疑', 
      source: 'free' 
    };
  }

  // 7. 合规性检查
  const complianceReport = await complianceCheckOrchestrator.performComplianceCheck({
    title: generated.title,
    content: generated.content,
  });
  
  if (!complianceReport.passed) {
    const reason = complianceReport.rejectReasons?.[0] || '未知原因';
    logger.warn(`自由发帖内容未通过合规检查：${reason}`);
    return { 
      success: false, 
      error: `内容未通过合规检查：${reason}`, 
      source: 'free',
    };
  }

  // 8. 多样化变换
  await this.applyDiversityTransformsForFreeStyle(generated);

  // 9. 素材选择
  const internetReferences: InternetReference[] = [];
  for (const ref of references) {
    const imageUrls = ref.processedImageUrls || ref.imageUrls || [];
    if (imageUrls.length > 0) {
      internetReferences.push({
        title: ref.title || '',
        content: ref.content || '',
        source: ref.source || 'internet',
        url: ref.url || '',
        imageUrls,
      });
    }
  }

  const materialSelectionResult = await hybridMaterialService.selectHybridMaterials({
    priorityMode: 'hybrid',
    localRatio: 0.6,
    title: generated.title,
    internetReferences: internetReferences.length > 0 ? internetReferences : undefined,
    neededCount: minImages,
  });
  
  const imagePaths = materialSelectionResult.selectedMaterials.map((m: any) => m.path);

  // 10. 上传图片
  const token = await this.authService.getAccessToken();
  let imageUrls: string[] = [];
  let currentMode = mode ?? (featuredEnabled ? 'featured' : 'normal');
  
  if (featuredEnabled) {
    imageUrls = await this.uploadImagesToMinCount(
      token, 
      imagePaths, 
      minImages,
      featuredConfig?.maxImageUploadRetries || 3
    );
    
    // 图片不足降级
    if (imageUrls.length < minImages && featuredEnabled) {
      logger.warn(`图片上传后数量不足（${imageUrls.length}/${minImages}），降级为普通帖`);
      currentMode = 'normal';
    }
  } else {
    const uploadResult = await this.api.uploadImages(token, imagePaths);
    imageUrls = uploadResult.urls;
  }

  // 11. 匹配话题
  const matchedTopics = await matchTopics(generated.title, generated.content, hotTopics);

  // 12. 发布帖子
  const response = await this.api.publishPost(
    token,
    generated.title,
    generated.content,
    {
      imageUrls: imageUrls.length > 0 ? imageUrls : undefined,
      topicList: matchedTopics.length > 0 ? matchedTopics : undefined,
    }
  );

  // 13. 记录日志
  if (response.success) {
    postLoggingService.log({
      timestamp: Date.now(),
      triggerType,
      postType: 'free',
      mode: currentMode,
      title: generated.title,
      content: generated.content,
      imageUrls,
      status: 'success',
    });
    
    return {
      success: true,
      postId: response.postId,
      title: generated.title,
      source: 'free',
      mode: currentMode,
    };
  }
}
```

```typescript
async function apiTokenMiddleware(req: any, res: any, next: any) {
  const authHeader = req.headers.authorization;
  
  // 检查 Authorization 头
  if (!authHeader || !authHeader.startsWith('Bearer ')) {
    return res.status(401).json({ 
      error: '缺少 Authorization 头', 
      code: 'UNAUTHORIZED' 
    });
  }

  const token = authHeader.split(' ')[1];
  
  // 特殊标记：配置文件中的 Token
  if (token === 'configured') {
    const status = await getTokenStatus();
    if (status.configured) {
      return next();
    }
  }

  // 验证 Token
  const isValid = await verifyApiToken(token);
  if (!isValid) {
    return res.status(401).json({ 
      error: 'Token 无效', 
      code: 'INVALID_TOKEN' 
    });
  }
  
  next();
}
```

```typescript
function rateLimitMiddleware(req: any, res: any, next: any) {
  const limit = 10;  // 每小时 10 次
  const windowMs = 60 * 60 * 1000;  // 1 小时

  const clientId = req.ip || req.headers['x-forwarded-for'] || 'unknown';
  const now = Date.now();

  let clientData = rateLimitStore.get(clientId);
```

#### 1.1 核心功能

2. **语义搜索**: 使用 ChromaDB 进行向量相似度匹配

#### 1.2 素材选择流程

```typescript
async selectHybridMaterials(options: {
  priorityMode?: PriorityMode;
  localRatio?: number;
  internetReferences?: InternetReference[];
  neededCount: number;
  title?: string;
}): Promise<MaterialSelectionResult> {
  
  // 1. 提取关键词
  if (options.title) {
    const keywords = await this.extractKeywordsFromTitle(options.title);
  }

  // 2. 匹配本地素材（语义搜索）
  const localMaterials = await this.matchLocalMaterials(keywords, neededCount);

  // 3. 处理互联网素材
  const internetMaterials = await this.processInternetReferences(
    options.internetReferences || []
  );

  // 4. 选择策略
  let selected: MaterialRecord[] = [];
  let strategy = '';

  if (priorityMode === 'local-first') {
    // 本地优先
    selected = [...localMaterials];
    if (selected.length < neededCount && internetMaterials.length > 0) {
      const needExtra = neededCount - selected.length;
      selected = selected.concat(internetMaterials.slice(0, needExtra));
      strategy = `本地优先：本地${localMaterials.length}个 + 网络补充${Math.min(needExtra, internetMaterials.length)}个`;
    }
  } else if (priorityMode === 'hybrid') {
    // ⭐ 混合模式：按贴合度排序，不强制混合
    
    const allCandidates = [...localMaterials, ...internetMaterials];
    
    if (allCandidates.length === 0) {
      strategy = '混合模式：无可用素材';
      selected = [];
    } else if (allCandidates.length <= neededCount) {
      // 候选不足，全部使用
      selected = allCandidates;
      const localCount = selected.filter(m => m.source === 'local').length;
      const internetCount = selected.filter(m => m.source === 'internet').length;
      strategy = `混合模式：候选不足，使用全部${selected.length}个（本地${localCount}+网络${internetCount}）`;
    } else {
      // 候选充足，按贴合度排序
      
      const scoredCandidates = allCandidates.map(material => {
        const matchScore = material.matchedKeywords ? 
          this.calculateMatchScore(material, options.title || '') : 0;
        const qualityScore = material.qualityScore?.totalScore || 60;
        
        // 综合分数 = 匹配度 70% + 质量 30%
        const totalScore = matchScore * 0.7 + qualityScore * 0.3;
        
        return { material, totalScore, matchScore, qualityScore };
      });
      
      // 按分数降序排序
      scoredCandidates.sort((a, b) => b.totalScore - a.totalScore);
      
      // 选择前 N 个
      selected = scoredCandidates.slice(0, neededCount).map(s => s.material);
      
      const localCount = selected.filter(m => m.source === 'local').length;
      const internetCount = selected.filter(m => m.source === 'internet').length;
      
      if (localCount === 0) {
        strategy = `混合模式：选择最贴合的${selected.length}个（全部网络素材）`;
      } else if (internetCount === 0) {
        strategy = `混合模式：选择最贴合的${selected.length}个（全部本地素材）`;
      } else {
        strategy = `混合模式：选择最贴合的${selected.length}个（本地${localCount}+网络${internetCount}）`;
      }
    }
  }

  return {
    selectedMaterials: selected,
    localCount: selected.filter(m => m.source === 'local').length,
    internetCount: selected.filter(m => m.source === 'internet').length,
    totalScore: selected.reduce((sum, m) => sum + (m.qualityScore?.totalScore || 50), 0),
    strategy,
  };
}
```

```typescript
export interface MaterialQualityScore {
  totalScore: number;      // 总分 (0-100)
  clarity: number;         // 清晰度 (0-20)
  composition: number;     // 构图 (0-20)
  lighting: number;        // 光线 (0-20)
  relevance: number;       // 相关性 (0-20)
  freshness: number;       // 新鲜度 (0-20)
}

async evaluateMaterialQuality(material: MaterialRecord): Promise<MaterialQualityScore> {
  let score: MaterialQualityScore = {
    totalScore: 50,
    clarity: 10,
    composition: 10,
    lighting: 10,
    relevance: 10,
    freshness: 10,
  };

  if (material.source === 'local') {
    // 本地素材：检查文件大小、分辨率等
    const stat = fs.statSync(material.path);
    const sizeKB = stat.size / 1024;
    
    // 清晰度：文件越大通常越清晰
    score.clarity = Math.min(20, Math.floor(sizeKB / 500));
    
    // 新鲜度：文件越新越新鲜
    const daysOld = (Date.now() - stat.mtimeMs) / (1000 * 60 * 60 * 24);
    score.freshness = Math.max(0, 20 - Math.floor(daysOld / 30));
    
  } else if (material.source === 'internet') {
    // 网络素材：默认给中等分数
    score.clarity = 15;
    score.composition = 15;
    score.lighting = 15;
    score.relevance = 15;
    score.freshness = 15;
  }
  
  score.totalScore = score.clarity + score.composition + score.lighting + 
                     score.relevance + score.freshness;
  
  return score;
}
```

```typescript
async performComplianceCheck(input: ComplianceCheckInput): Promise<ComplianceCheckResult> {
  const result: ComplianceCheckResult = {
    reportId: this.generateReportId(),
    passed: true,
    rejectReasons: [],
    qualityScore: undefined,
  };

  // 1. 内容去重（ChromaDB 语义相似度）

  );

    result.rejectReasons.push(

  const enhancedResult = await enhancedSensitiveWordService.detectAndReplace(input.content);

  const qualityScore = await contentQualityScoringService.calculateScore({
    title: input.title,
    content: input.content,
    imageCount: input.imageCount || 0,
    topicNames: input.topicNames || [],
  });
  
  result.qualityScore = qualityScore;
  
  if (qualityScore.finalScore < 60) {
    result.passed = false;
    result.rejectReasons.push(`质量评分过低（${qualityScore.finalScore}分）`);
  }

  // 4. 发帖间隔检查
  if (input.topicId) {
    const intervalResult = await postingIntervalControlService.checkPostingInterval(
      input.topicId
    );
    
    if (!intervalResult.allowed) {
      result.passed = false;
      result.rejectReasons.push(
        `发帖间隔不足（需间隔${intervalResult.requiredIntervalHours}小时）`
      );
    }
  }

  return result;
}
```

```typescript
const rateLimitStore = new Map<string, { count: number; resetTime: number }>();

function rateLimitMiddleware(req: any, res: any, next: any) {
  const limit = 10;  // 每小时 10 次
  const windowMs = 60 * 60 * 1000;  // 1 小时

  const clientId = req.ip || req.headers['x-forwarded-for'] || 'unknown';
  const now = Date.now();

  let clientData = rateLimitStore.get(clientId);

    rateLimitStore.set(clientId, clientData);

  if (clientData.count >= limit) {
    return res.status(429).json({ 
      error: '请求频率超限', 
      code: 'RATE_LIMIT_EXCEEDED',
      retryAfter: Math.ceil((clientData.resetTime - now) / 1000)
    });
  }

  clientData.count++;
  next();
}
```

## 关键设计决策

### 1. 为什么使用互联网参考？

**问题**: 自由发帖时，AI 生成的内容可能缺乏真实感和时效性

**解决方案**: 
- 从小红书、微博、知乎等真实社区获取参考素材
- AI 基于真实案例进行改写和再创作
- 保证内容的真实性和时效性

**优势**:
- ✅ 内容更贴近真实用户体验
- ✅ 紧跟热点和流行话题
- ✅ 避免 AI 生成内容的空洞感

### 2. 为什么需要频率限制？

**问题**: 频繁调用搜索 API 可能触发平台反爬机制

**解决方案**:
- 每小时最多查询 10 次（可配置）
- 使用内存计数器简单高效
- 超限后拒绝查询

**优势**:
- ✅ 避免 IP 被封禁
- ✅ 分散请求压力
- ✅ 符合平台使用规范

### 3. 为什么使用平台轮询？

**问题**: 单一平台可能不稳定或搜索结果有限

**解决方案**:
- 支持 4 个平台（小红书、微博、知乎、汽车之家）
- 每次查询自动切换平台
- 首选平台失败后自动降级

**优势**:
- ✅ 提高搜索成功率
- ✅ 分散请求到多个平台
- ✅ 获取更丰富的参考素材

### 4. 为什么按贴合度排序选择素材？

**问题**: 强制混合本地和网络素材可能导致质量下降

**解决方案**:
- 不强制要求两种素材同时存在
- 按贴合度（匹配度 70% + 质量 30%）排序
- 选择最贴合的前 N 个素材

**优势**:
- ✅ 保证素材质量优先
- ✅ 避免为混合而混合
- ✅ 更灵活的素材选择策略

### 5. 为什么合规检查异常时允许发布？

**问题**: 合规检查服务异常可能导致无法发帖

**解决方案**:
- 检查异常时记录日志
- 降级处理，允许发布
- 保证核心功能可用性

**优势**:
- ✅ 提高系统可用性
- ✅ 避免单点故障
- ✅ 异常情况仍可发帖

### 2. 批量上传优化

```typescript
async uploadImagesToMinCount(
  token: string,
  candidates: string[],
  minImages: number,
  maxRetries: number
): Promise<string[]> {
  const remaining = [...candidates];
  const used = new Set<string>();
  const imageUrls: string[] = [];

  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    if (imageUrls.length >= minImages) break;
    
    const available = remaining.filter(p => !used.has(p));
    if (available.length === 0) break;
    
    const need = minImages - imageUrls.length;
    const batchSize = Math.min(9, available.length, Math.max(minImages, need));
    const batch = available.slice(0, batchSize);
    
    for (const p of batch) used.add(p);
    
    const uploadResult = await this.api.uploadImages(token, batch);
    for (const url of uploadResult.urls) {
      if (!imageUrls.includes(url)) imageUrls.push(url);
    }
  }

  return imageUrls;
}
```

**优化效果**:
- ✅ 批量上传，减少 API 调用次数
- ✅ 失败重试，提高成功率
- ✅ 避免重复上传

## 常见问题排查

### 1. 互联网参考查询失败

**现象**: 自由发帖时提示"互联网参考查询未返回结果"

**排查步骤**:
1. 检查互联网参考服务是否启用
   ```sql
   SELECT enabled FROM internet_reference_config LIMIT 1;
   ```

2. 检查可用平台
   ```sql
   SELECT platform_name, enabled, priority FROM internet_reference_platforms;
   ```

3. 查看搜索日志
   ```
   搜索关键词：互联网参考素材查询
   错误信息：平台 xxx 搜索失败
   ```

**解决方案**:
- 启用互联网参考服务
- 添加或启用更多搜索平台
- 检查网络连接

### 2. 频率限制超限

**现象**: 提示"互联网参考查询频率超限"

**排查步骤**:
1. 查看当前配置
   ```sql
   SELECT rate_limit_per_hour FROM internet_reference_config;
   ```

2. 查看查询计数（日志）
   ```
   搜索关键词：互联网参考查询频率
   ```

**解决方案**:
- 等待一小时后自动重置
- 提高 `rateLimitPerHour` 配置
- 优化查询策略，减少不必要的查询

### 3. 素材选择失败

**现象**: 提示"混合素材选择失败"

**排查步骤**:
1. 检查本地素材库
   ```sql
   SELECT COUNT(*) FROM material_records WHERE status = 'available';
   ```

2. 检查 ChromaDB 连接
   ```
   搜索关键词：【语义搜索】
   ```

3. 查看素材匹配日志
   ```
   搜索关键词：混合模式：
   ```

**解决方案**:
- 添加更多本地素材
- 确保 ChromaDB 服务正常运行
- 检查素材文件路径是否正确

### 4. 抄袭检测误判

**现象**: 原创内容被判定为抄袭

**排查步骤**:
1. 查看抄袭检测报告
   ```typescript
   const isPlagiarism = detectPlagiarism(generated.content, references);
   ```

2. 检查参考素材
   ```
   搜索关键词：互联网参考素材
   ```

**解决方案**:
- 调整 AI 改写策略，增加原创性
- 使用更多参考素材，避免单一来源
- 优化抄袭检测算法
